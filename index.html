HTTP API

简洁的 REST 接口

三类接口覆盖搜索、文档管理、用户行为上报

POST/api/v1/search全文搜索，支持个性化
GET/health健康检查，返回索引统计
POST/api/v1/doc/add实时添加文档，立即可搜
PUT/api/v1/doc/update更新文档元数据
DELETE/api/v1/doc/delete删除文档
POST/api/v1/event/click上报点击行为，写 events.db + 更新画像
POST/api/v1/event/like上报点赞，更新用户 category_weights
POST/api/v1/event/dwell上报停留时长，生成弱正样本
POST/api/v1/event/dismiss上报负反馈，生成负样本
POST/api/v1/admin/reload_model双 Buffer 热更新，不停机切换 LightGBM 模型
